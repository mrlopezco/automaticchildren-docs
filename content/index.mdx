# Automatic Children Docs

Welcome to the documentation for the **Automatic Children** Azure DevOps extension.

**Automatic Children** lets you define rules to automatically create child work items when you run the action from a work item's context menu. You configure which parent types and states apply, optional field conditions, and the child work item type and field mappings—including placeholders from the parent so titles and fields stay in sync.

This site hosts all end-user documentation: setup, configuration, and usage guides.
